# Dishly Frontend

Frontend en React + TypeScript per a la gestió de menús, plats i al·lèrgens.

## Setup

```bash
npm install
npm run dev
```
Si dona conflicte de versions entre Vite i React, prova: 
```bash
 npm install --legacy-peer-deps
```
Instalar rect icons:
```bash
npm install react-icons --legacy-peer-deps
```


## Backend

Aquest frontend es connecta a una API REST + WebSockets.

## Stack

* React
* TypeScript
* Vite
* Socket.io
